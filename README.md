# Lead-IQ Assignment
Create a single-page application that allow user to search a GitHub user and view their repositories + render README file.

## Main stack
- react
- react-router-dom
- webpack
- axios
- less
- momentjs
- redux
- redux-observable
- rxjs
- react-markdown

## Get Started
```
npm install && npm run dev
```
